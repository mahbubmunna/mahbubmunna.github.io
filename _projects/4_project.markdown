Features  of Ideal Flix

- User registration
- User profile
- Subscription and payment gateway
- Content search
- Movie categories(thriller, horror, comedy)
- Play subtitles and change languages
- Movie resolutions upto 2100 P.
- Notifications
- Reviews and rating
- Customize design
