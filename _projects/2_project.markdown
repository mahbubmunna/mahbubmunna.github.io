Hungry Hub is an food delivery application with following features 

- Login & Registration
- One Tap Onboarding
- Find & Select Restaurant
- Browse Menu
- Customize Orders
- View Cart
- Push Notifications & Alerts
- Secured Payment
- Social Media Integration
- Offers & Promos
- Provide Special Instructions
- Contact Delivery Agent
- Track Delivery Agent
- Tip Delivery Agent
- Favourite Orders
- Food Details
- Repeat Orders
- Cancel Orders
- Track Order Status
- Filter By Cuisine
- Filter Restaurants
- Rate & Review Food
- Rate Delivery Agent
- Help & Support
- Report Issues
